# AmazonVentasNuevo
Este conjunto de datos ofrece una visión clara y estructurada del rendimiento de las ventas de Amazon. Se utilizó para crear un panel completo de Power BI que permite realizar un seguimiento de las métricas clave del negocio. Métricas clave analizadas: Ingresos totales: 32,87 millones; Beneficio total: 1,64 millones
